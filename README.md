# BremenPV 
### A feasibility study for Bremen’s PV self-sufficiency on heating consumption – a demo in an old Bremen district

<br>

**Methods**
-----
- Integrated approach to incorporate outputs from geometrical modules (in FreeCAD) with geographical information system (GIS) tools to analyze the feasibility of PV contribution by single households based on the building rooftop area.
